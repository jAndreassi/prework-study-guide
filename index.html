CLI

  • ls - A list of files and directories
  • cd - Change directories(ex- cd downloads - navigates into downloads)
  • pwd - print working directory
  • cd .. - navigates backwards one directory at a time
  • mkdir - Make directory
  • touch - create new file
  • start - open the file

5 steps of debugging

  • 1) What the code is doing as opposed to what the code is actually doing.
  • 2) Hunt for the bug within the code to see what is actually causing the problem.
  • 3) After the bug has been discovered try implementing different solutions to fix the bug.
  • 4) Once you have implemented a fix, test the app to make sure changes are effective and no new bugs were added. There are different tools and procedures for tesing a bug fix, will need to choose the best tool and process to help fix the project.
  • 5) Document process used to achieve solution. Can help to refer back if same issue happens again.

HTML

  • HTML - Hyper Text Markup Language
  • Elements - adds contents to page. (Text elements, Image elements, Header elements, and List elements)
  • Elements - to divide and describe a page. (Meta-related elements, Content division elements, and Semantic elements)
  • HTML attributes are always defined in the begining element, never the closing element.
  • < head > element -info or metadata that renders page correctly. Not visable to user on webpage.
  • < body > contents shown to user.
  • < style > element defines the CSS style associated with a website.
  • < base > element defines the base for URL for a webpage.
  • < link > element connects an external resource to the HTML document.
  • < meta > element defines metadata such as the character set, description, keywords, author, and viewpoint. Contains info about the page that is used by the browser.
  • < script > element embeds client side scripts in the HTML document
  • < ul > unordered list, < ol > ordered list, < img > image, < src > location of image, < a > anchor, < href > points to the URL link.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside am element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer excutes code in script.
